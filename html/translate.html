快捷键说明
Win + Alt + Q
(全局)启动翻译器，自动将当前选中的高亮文本填入翻译输入框。
Ctrl + Tab
切换焦点至翻译输入框。
Ctrl + `
切换翻译页面和历史记录。
Ctrl + 空格
触发翻译（等同于点击“翻译”按钮）。
Ctrl + O
窗口半透明切换。
Ctrl + T
窗口置顶切换。
